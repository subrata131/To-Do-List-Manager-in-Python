📝 **To-Do List Manager in Python (CLI + File Handling)**

Built a simple yet practical **To-Do List application** using Python that helps manage daily tasks efficiently. This project focuses on clean logic, user interaction, and data persistence.

---

### 🚀 **Features**

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* 💾 Save tasks permanently using file handling

---

### 🧠 **Concepts Used**

* Python Lists (Data Structure)
* File Handling (`read/write`)
* Functions (Modular Programming)
* Loops & Conditional Statements
* Exception Handling (`try-except`)
